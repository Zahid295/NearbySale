### NearbySale - Full Stack web application for Furniture Business
### Technologies used 
This project utilizes Flask web framework for backend, PostgreSQL and Flask-SQLAlchemy for databases and Jinja Template language, HTML, CSS for frontend and Flask-Admin for admin managemenet of this full stack web application. The project is also deployed on Heroku cloud platform. 

NearbySale is an online furniture e-commerce application that will serve the growing demand for furniture sales online. The "NearbySale" software strives to close the gap between customers and furniture sellers by offering a smooth online purchasing experience.
